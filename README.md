<h2>A Little About Ranie</h2>

<p>Ranie is a technology for businesses that run chains of warehouses or supermarkets.</p>

<p>The project was built for <a href="https://erp.ranie.com.ng/admin/warehouse">Ranie</a></p>

<p>The Ranie technology has all needed features for such businesses, including Inventory Management, Customer Management, Location Management, Sales Management, Point of Sale (for payment) System, etc.</p>

<h2>Staff Role Management</h2>

<p>Ranie lets the company manage all aspects of their business, including assigning roles and permissions to all employees in such a way that an employee can only do what is permitted by their access level within the system. This ensures security and privacy</p>

<h2>Multiple Locations</h2>

<p>The Ranie technology lets the business manage operations happening in multiple locations in Nigeria (from Lagos to Abuja) from one single dashboard, filter statistics based on each warehouse/supermarket, aggregate statistics summary from all warehouses/supermarkets, etc.</p>
